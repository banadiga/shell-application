## Requroments

Build project [https://github.com/banadiga/client-bank](https://github.com/banadiga/client-bank) before

## Build

```
mvn clean install
```

## Run 
```
mvn spring-boot:run
```
or
```
java -jar target/shell-application-0.0.1-SNAPSHOT.jar 
```

## Connect via SSH

```
ssh -p 2222 user@localhost
```


Use any password to login
